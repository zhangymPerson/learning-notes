# vscode的配置文件说明

>vscode 有三个配置文件 launch.json  setting.json tasks.json
- [官方文档](https://code.visualstudio.com/docs)
## launch.json

- 主要是进行运行环境的相关配置

    可以自动生成 - 点击左侧的第四个debug/run按钮(快捷键 ctrl+shift+d)创建 launch.json 文件

    这个文件中配置启动相关命名等

- 执行一些命令的配置

## setting.json

- vscode的项目相关的定制化配置

## tasks.json

- 编译过程的配置等

## 区别

- tasks可以被用来做编译，而launch用来执行编译好的文件。
