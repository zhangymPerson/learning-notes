- 定义

  Gradle 是一个基于 JVM 的构建工具，是一款通用灵活的构建工具，支持 maven， Ivy 仓库，支持传递性依赖管理，而不需要远程仓库或者是 pom.xml 和 ivy.xml 配置文件，基于 Groovy，build 脚本使用 Groovy 编写。
