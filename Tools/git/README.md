## git分支的理解

- 分支

    git有一个主分支 master

    可以创建多个分支 并且 任意命名

    可以任意合并两个分支

    分支之间无任何父子关系， 之所以有dev-等分支 是人为约定， 任何分支可以不合并到主分支 可以随便切换所需分支

    开发组可以定义自己的分支之间逻辑，但是在git层面上，所有分支之间无特定关系 都是平级的
