- 浏览器插件使用方法

    下载 浏览器插件文件  .crx文件

    修改文件后缀为zip,新建同名文件夹,将该zip文件放到文件夹下并解压

    在谷歌浏览器中输入  chrome://extensions/ 打开开发者模式 

    加载已解压的扩展程序 选择zip文件所在的文件夹即可安装成功

    重启浏览器
