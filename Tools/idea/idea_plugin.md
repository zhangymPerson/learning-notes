## idea-plugin

### vim插件

- 名称 ideaVim

- 安装 

    点击 File > settings - > plugins > 搜索vim > IdeaVim 

- 配置

    用户目录下添加配置文件
    C:\Users\Administrator

    创建 .ideavimrc文件

    不能直接创建.**的文件  可以cmd / powershell

    windows 下 配置文件 .ideavimrc 和 _ideavimrc 均可

    ```sh
    echo a > .ideavimrc 
    ```
