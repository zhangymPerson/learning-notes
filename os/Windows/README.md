## cmd 使用

- 查看环境变量

  快捷键 `win+r` - `cmd` - `path` 即可查看

  查看指定的变量 `set JAVA_HOME`

## powershell 使用

- 查看环境变量的命令

  powershell : `Get-ChildItem env:`

  查看指定的 ： `$env:JAVA_HOME`
