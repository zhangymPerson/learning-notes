# 数据库相关的软件

## DB 数据库管理软件

- PDMan

  免费开源的数据建模软件

  gitee：<https://gitee.com/robergroup/chiner>

  下载发型版本地址：<https://gitee.com/robergroup/chiner/releases>

- DBeaver

  免费开源的数据库连接工具

  官网：<https://dbeaver.io/>

  github:<https://github.com/dbeaver/dbeaver>

  vim 插件安装
  
  菜单 -> Help -> Install New Software 添加 `http://vrapper.sourceforge.net/update-site/stable` 安装 `vrapper` 插件
